This is the website of an online store. In it, you can choose categories by gender as well as by type of clothing. 
There is a shopping cart and a favorites section. The site is bundled with a simple back-end based on JSON Server package. 
React Router Dom was used in the project. 
After downloading the code, you should have Vite and npm installed. You should also install React Router Dom.
This can be done using commands in the terminal:
npm install
npm install react-router-dom
You can create a project in Vite using:
npm create vite@latest
